Chainer

Enter multiple formulas into the text area below. Parsing (plus conversion into NNF) will happen automatically and the interpreted inputs will appear in the list below. Accepted syntax contains atomic propositions [a-z], propositional constants T (true), F (false) and logical connectives ^ (negation), * (conjunction), + (disjunction), -> (implication). If the inputs are valid, Derive in PSC will build a tree of deduction chains, where each leaf is a separate D-chain. Special sequence items are distinguished formulas, tautologies and leaf identifiers axiom and irreducible. If every leaf is an axiom, the input sequence is derivable in PSC.

Example input: ((a*T)+c)+(^c*(b+F))
